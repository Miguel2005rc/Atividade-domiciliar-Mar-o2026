# Atividade-domiciliar-Mar-o2026
Atividade domiciliar Março2026

Este é uma atividade de um sistema para gerenciamento de protocolos .

##  Objetivo
Permitir que usuários organizem suas atividades , tendo protocolos e processos para custo de viagens , quantidade de diarias e prazos no trabalho , acompanhando o progresso

 Custos de viagens
 Quantidade de diárias
 Definição de prazos no trabalho
 Acompanhamento do progresso das atividades

##  Funcionalidades
Cadastro de protocolos
 Controle de despesas de viagem
 Registro de diárias
 Definição e acompanhamento de prazos
 Monitoramento do andamento das atividades
